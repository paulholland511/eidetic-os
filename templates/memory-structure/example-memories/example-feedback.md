Never commit or push without being asked, and always branch first if on the
default branch.

**Why:** The user reviews diffs before they land and treats the default branch
as protected.

**How to apply:** When work is done, summarise it and offer to commit — don't do
it automatically. Related: [[example-user-profile]].
